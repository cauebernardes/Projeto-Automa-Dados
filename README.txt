AUTOMA DADOS - DEMO PORTAVEL (STREAMLIT)

IMPORTANTE:
- Edite o arquivo .env e coloque sua GEMINI_API_KEY antes de iniciar.
- O banco local desta demo e: projeto.sqlite (na mesma pasta).

Arquivos principais:
- app.py
- projeto.sqlite
- .env
- requirements.txt
- INICIAR_SISTEMA.bat

Como executar:
1) Abrir .env e preencher GEMINI_API_KEY.
2) Dar duplo clique em INICIAR_SISTEMA.bat.
3) Acessar http://localhost:8501.

Obs. de seguranca:
- Nao compartilhar a chave de API com o grupo.
